# JavaScript Object Access

## Access Value, Nested Object, Optional Chaining, Dot vs Bracket Notation

---

## 📌 Topic Overview

এই প্র্যাকটিস সেটে আমরা JavaScript Object থেকে data access করার সব গুরুত্বপূর্ণ প্যাটার্ন **practice-driven** ভাবে শিখবো।

Covered Concepts:

- Object value access
- Dot notation vs Bracket notation
- Special property names access
- Numeric keys
- `for...in` loop
- Nested object access
- Optional chaining (`?.`)
- Dynamic property access

---

## 1️⃣ Concept Analysis

### 🔹 Core Concepts to Learn

- Object এর property কীভাবে access করতে হয়
- Dot notation কখন কাজ করে
- Bracket notation কখন mandatory
- Numeric key access
- Special character (`-`) থাকা key access
- Variable ব্যবহার করে property access
- `for...in` দিয়ে object traversal
- Nested object structure
- Undefined property access error
- Optional chaining দিয়ে safe access

---

### 🔹 Concepts Used in Given Code

- `student.name` → dot notation
- `student['name']` → bracket notation
- `student['home-address']` → special key access
- `student[propName]` → dynamic access
- `for (const key in student)` → iterate object
- `student.family.title` → nested object
- `student.family.mother?.age` → optional chaining

```js

};
```

**Hint:**

- Object key internally string হয়

**Concept:**

- Numeric key access
- Bracket notation

### Q4
